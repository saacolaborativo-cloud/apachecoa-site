# ApachecoA

Sitio web de ApachecoA: espacio de experimentación artística, producción teórica y proyectos colaborativos enfocados en prácticas críticas, comunitarias y experimentales.

## Descripción

ApachecoA es una plataforma donde se reúnen textos, obra y procesos colaborativos que exploran la relación entre arte, teoría y pensamiento crítico. El sitio funciona como archivo y espacio vivo en constante desarrollo.

## Contenido

- Semblanza
- Enunciado de artista
- Sa’a Colaborativo
- Textos (ensayos, artículos de opinión, críticas, poemas, libros)
- Obra (video, imagen y proyectos)
- Fogón Radio
- Contacto

## Estado del proyecto

En desarrollo continuo. El contenido se irá actualizando y ampliando con el tiempo.

## Tecnologías

- HTML
- CSS
- GitHub Pages (para publicación)
